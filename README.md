# 🌐 RestTemplate Integration in Spring Boot

📌 Overview
==============

This project shows how to use Spring Boot RestTemplate to:

✅ Send POST request (Create User)

✅ Send GET request (Fetch User)

✅ Work with Path Variables

✅ Handle API responses

**🧠 Key Concepts**
======================
RestTemplate → Used to call APIs

HttpEntity → Sends request body

ResponseEntity → Stores response

HttpMethod → Defines GET / POST

Map → Pass path variables

**🔄 Methods Used**
=====================
Method	           Purpose
======            ==========
POST	            Create user
GET               Fetch user


**🚀 How It Works**
=====================
Spring Boot App → RestTemplate → External API → Response Back
